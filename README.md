# foam_tools

OpenFOAM 向けの小さなユーティリティ集です。各ツールは専用フォルダにスクリプトと README があります。

| フォルダ | 内容 |
|----------|------|
| [foam_velocity_slice](foam_velocity_slice/) | 座標平面での流速スライス画像出力 |
| [foam_velocity_slice_position](foam_velocity_slice_position/) | 任意の点を通る平面での流速スライス |
| [foam_log_plot](foam_log_plot/) | foamLog の残差を1枚のグラフにまとめる |

各ディレクトリの README に依存パッケージと実行例を記載しています。
